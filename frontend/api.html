🎭 換臉 API

核心的 AI 換臉功能

POST /api/face-swap

執行 AI 換臉操作，支援預設模板和自訂模板。

請求參數

參數名 類型 必填 描述
file File 必填 使用者上傳的照片檔案
template_id String 必填 模板 ID (1-6) 或 "custom"
template_file File 選填 自訂模板檔案（當 template_id="custom" 時）
source_face_index Integer 選填 來源圖片中的臉部索引（預設: 0）
target_face_index Integer 選填 模板圖片中的臉部索引（預設: 0）

📝 請求範例

curl -X POST "http://localhost:8882/api/face-swap" \
  -F "file=@user_photo.jpg" \
  -F "template_id=1" \
  -F "source_face_index=0" \
  -F "target_face_index=0"

回應格式

200 OK 成功回應：

{
  "success": true,
  "message": "換臉完成！",
  "result_url": "/results/result_abc123.jpg",
  "template_id": "1",
  "template_name": "模板 1",
  "template_description": "預設模板"
}

400 Bad Request 錯誤回應：

{
  "detail": "檔案格式不支援"
}

🔍 圖片驗證 API

驗證圖片並獲取臉部資訊

POST /api/validate-image

驗證上傳的圖片是否包含可識別的臉部。

請求參數

參數名 類型 必填 描述
file File 必填 要驗證的圖片檔案

回應格式

200 OK 成功回應：

{
  "success": true,
  "message": "圖片驗證成功",
  "image_info": {
    "valid": true,
    "face_count": 2,
    "image_size": [1920, 1080]
  }
}

❌ 錯誤代碼

常見的 API 錯誤代碼和處理方式

狀態碼 錯誤類型 描述 解決方案
400 Bad Request 請求參數錯誤或檔案格式不支援 檢查請求參數和檔案格式
413 Payload Too Large 上傳檔案過大 壓縮圖片或使用較小的檔案
415 Unsupported Media Type 不支援的檔案格式 使用 JPG、PNG 或 WebP 格式
500 Internal Server Error 伺服器內部錯誤 檢查伺服器日誌或聯繫管理員

⚠️ 使用限制

API 使用的限制和注意事項

  • 檔案大小限制：最大 10MB
  • 支援格式：JPG、JPEG、PNG、WebP
  • 臉部數量：最多支援 20 張臉部
  • 結果保留：處理結果保留 24 小時
  • 上傳檔案：臨時檔案保留 1 小時
  • 並發限制：建議同時處理不超過 5 個請求
